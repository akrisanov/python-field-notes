# Learning Path

A practical route through this repository.

1. Start with `notes/tutorial/00_intro.ipynb` and continue sequentially.
2. Deepen specific topics in `notes/packages/`.
3. Practice quick patterns in `notes/tips_and_tricks/`.
4. Work through assignment-oriented tasks in `exercises/coursera/`.
5. Use runnable scripts in `examples/` for focused experiments.
